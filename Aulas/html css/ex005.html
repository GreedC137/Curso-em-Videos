Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica)

Nesta frase tenho um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos também fazer um texto marcado com a tag MARK

Texto grande e texto pequeno

Neste parágrafo criaremos um texto grande e um texto pequeno com as tags BIG e SMALL

Texto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido mas não considerado com a tag DEL

Texto incluido

Podemos marcar um texto inserido para dar ênfase e indicar que ele foi adicionado com a tag INS

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Serve para coisas do tipo X20 + 50, utilizamos a tag SUP

Texto subscrito

Serve para coisas do tipo H2O, utilizando a tag SUB
